# Mini LMS System 🎓

A responsive multi-page Learning Management System (LMS) web application built using frontend technologies.

## 🚀 Features
- Student Login & Signup Interface
- Course Listing and Detail Pages
- Interactive Dashboard Layout
- Multi-page Navigation System
- Clean and Responsive UI Design

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript 

## 🎯 Purpose
This project was developed to strengthen frontend development skills, UI design understanding, and multi-page website structure.
